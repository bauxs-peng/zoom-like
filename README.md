# zoom-like

Powerful zoom-like tool

## Description
This is a powerful tool that replicates the functionalities of Zoom, offering features such as video conferencing, screen sharing, and chat.

## Features
- Video conferencing
- Screen sharing
- Chat functionality

## Tech Stack
- JavaScript
- HTML
- CSS

## Usage
To install and run this project, clone the repository and open it in your browser.
